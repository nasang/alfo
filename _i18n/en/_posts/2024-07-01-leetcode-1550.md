### 1550. Three Consecutive Odds <font color="green">(Easy)</font>

```py
# Time: O(N)
# Space: O(1)
class Solution:
    def threeConsecutiveOdds(self, arr: List[int]) -> bool:
        n_cons = 0
        for x in arr:
            if x % 2 == 1:
                n_cons += 1
                if n_cons == 3:
                    return True
            else:
                n_cons = 0
        return False
```
