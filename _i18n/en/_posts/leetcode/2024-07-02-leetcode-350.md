### 350. Intersection of Two Arrays II <font color="green">(Easy)</font>

#### Initial Solution

```py
# Time: O(M + 2N) / O(2M + N)
# Space: O(M + N)
class Solution:
    def intersect(self, nums1: List[int], nums2: List[int]) -> List[int]:
        c_nums1 = Counter(nums1)
        c_nums2 = Counter(nums2)
        res = []
        for x1, c1 in c_nums1.items():
            c2 = c_nums2.get(x1, 0)
            cnt = min(c1, c2)
            if cnt > 0:
                res.extend([x1] * cnt)
        return res
```

#### Optimized

We only need to count the smaller array to save time and space.

```py
# Time: O(M + N)
# Space: O(min(M, N))
class Solution:
    def intersect(self, nums1: List[int], nums2: List[int]) -> List[int]:
        if len(nums1) > len(nums2):
            return self.intersect(nums2, nums1)
        c_nums1 = Counter(nums1)
        res = []
        for x in nums2:
            if c_nums1.get(x, 0) > 0:
                c_nums1[x] -= 1
                res.append(x)
        return res
```

#### Follow up

1. What if the given array is already sorted? How would you optimize your algorithm?

```py
# Time: O(NlogN + MlogM) for sorting; O(min(M, N)) for sorted
# Space: O(1)
class Solution:
    def intersect(self, nums1: List[int], nums2: List[int]) -> List[int]:
        # -- Follow up 1 --
        nums1.sort()
        nums2.sort()
        # -----------------

        i, j = 0, 0
        m, n = len(nums1), len(nums2)
        res = []
        while i < m and j < n:
            if nums1[i] == nums2[j]:
                res.append(nums1[i])
                i += 1
                j += 1
            elif nums1[i] < nums2[j]:
                i += 1
            else:
                j += 1
        return res
```
